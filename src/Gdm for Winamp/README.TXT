Gdm for Winamp u1r0 by Brendon, 02/07/2015.


——About
A bundle of EQF files each defining an equalizer (EQ) preset made with and for
Winamp, a well-known media player before VLC.

See `GDM.TXT` for complete project documentation. It is not included in this
package, but can be found in the project's main directory.


——Presets
Gdm Essential   Essential bass and treble enhancements for everyday listening.
Gdm Standard    The standard Gdm enhancement curve.
Gdm Extended    Standard plus mid-bass enhancements.
