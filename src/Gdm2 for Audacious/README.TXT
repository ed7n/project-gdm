Gdm2 for Audacious u2r0 by Brendon, 05/08/2017.


——About
The package containing this file includes a bundle of equalizer preset files
made with and for Audacious, a popular media player alternative to Winamp.

See `GDM.TXT` for complete project documentation. It is not included in this
package, but can be found in the project's main directory.


——Presets
Gdm2 Essential  Essential bass and treble enhancements for everyday listening.
Gdm2 Standard   The standard Gdm enhancement curve.
Gdm2 Extended   `Standard` plus mid-bass enhancements.
Gdm2 Bass       Bass boost.
Gdm2 Treble     Treble boost.
